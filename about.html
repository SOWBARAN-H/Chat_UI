About Chat

Chat is a modern, retro-inspired chat interface built with HTML and Tailwind CSS. It brings together classic chat vibes and contemporary design for a fun, easy-to-use experience.

Features

  • Responsive design
  • Classic chat layout
  • Modern UI elements
  • Easy navigation
  • Contact list

Technologies

  • HTML5
  • Tailwind CSS
  • Accessible markup

Story

Inspired by classic chat apps, Chat brings a nostalgic feel with a modern twist. Whether you're chatting with friends or exploring UI design, this project is a showcase of simplicity and style.
