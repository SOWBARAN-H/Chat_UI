
# Chat UI (WhatsApp 2019 Inspired)

This project is a modern chat interface inspired by WhatsApp's 2019 desktop layout, built with HTML and Tailwind CSS. The UI features a sidebar for chats, a main chat area, and a purple/blue gradient color scheme for a unique look.

## Features
- Responsive design for desktop and mobile
- Sidebar chat list with avatars and preview messages
- Modern header and navigation
- Distinct chat bubbles for sent/received messages
- Stylish contact list
- Accessible markup

## Project Structure
- `index.html` — WhatsApp 2019-inspired chat interface (Alex, Sara, Liam)
- `contacts.html` — Modernized contact list with avatars and details
- `about.html` — Project overview, features, technologies, and story

## Technologies Used
- HTML5
- Tailwind CSS (via CDN)

## How to Use
1. Clone or download this repository.
2. Open any HTML file in your browser.
3. Explore the chat UI, switch between chats, and view contacts/about pages.

## Customization
- Add more contacts or chats by editing the HTML files.
- Change colors and styles easily using Tailwind CSS utility classes.
